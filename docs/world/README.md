# world/ — the IP: persona craft, canon, the worlds

The persona & world-writing workstream. The prompts are the IP; this is where the craft, the shared canon, and each world's design live.

**Structure:** the root holds the **generic documents that apply to every world**; **each world gets its own folder** underneath, with its own docs and README.

## Generic (root — applies to every world)

| Doc | What it is |
| --- | ---------- |
| **[worlds.md](worlds.md)** | The district register: every world we've discussed (Switchboard/World Zero, the Underworld, the Kitchen, 2036, the divine, the Phreak district, the Philosophical realm) with status — building / dormant / candidate / seed. The map. |
| **[canon.md](canon.md)** | The world's shared book of record: structured, ratifiable entries (the Exchange, the Management, the Ledger, the Underworld, the Devil's personnel file, …). Every persona draws from it, so the universe stays coherent. Postcard-sized; grows via the transcript loop. |
| **[persona-design-process.md](persona-design-process.md)** | The repeatable method for designing an Elsewhere persona — the layered assembly (canon / character / craft), the cold-opener law, the principles. Refined every run. |
| **[persona-design-notes.md](persona-design-notes.md)** | The lab journal: dated learnings from real calls (what each transcript taught us). Newest first. Feeds the process doc. |

## Worlds (one folder each)

| Folder | World | Status |
| ------ | ----- | ------ |
| **[switchboard/](switchboard/)** | The Switchboard — World Zero (the hub, the Operator, the teaser) | 🔨 designed, build pending |
| _(future)_ underworld/ | The Underworld — the Devil bible + Underworld canon | 🟡 canon v1 in redline; bible next |

New worlds get a folder here as they graduate from a seed in `worlds.md` into real design.

The flywheel: read transcripts → find the real FAQ / thin walls → write tested material into the prompts and canon → every persona knows it next call. Sven is editor-in-chief; git review is the ratification gate.
