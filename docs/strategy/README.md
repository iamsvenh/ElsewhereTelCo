# strategy/ — what this is, where it goes, how it pays

The strategy & concept workstream. What Elsewhere is, why it wins, how it makes money, how it gets funded, how it launches.

| Doc | What it is |
| --- | ---------- |
| **[concept.md](concept.md)** | The operational source of truth: personas, legality, unit economics, staged roadmap, kill criteria. The index of record — read before any product decision. |
| **[vision-world-company.md](vision-world-company.md)** | The vision (v2, ratified). Elsewhere as an interactive-fiction *world company*: locked terminology, cosmology, the ontology of the medium, production layers, trajectory. The frame everything else fits inside. |
| **[business-model.md](business-model.md)** | How value is captured: revenue streams (venue B2B, phone-plan subscription, calling cards, hardware/merch, in-lore native sponsorship, licensing), the pricing fiction, and unit economics / cost structure. |
| **[fundraising.md](fundraising.md)** | The funding thesis: media-company comps (Gimlet lineage), strategic partners and investor types, the artifact that raises (retention + video), and when to raise. |
| **[go-to-market.md](go-to-market.md)** | Pre-launch traction and the pitch deck(s): the mystery-as-marketing tension, channels, the multiple deck versions, and the discipline gate (don't outrun the sharing signal). |
| **[outreach-log.md](outreach-log.md)** | Running log of every deliberate push into a community/channel — exact copy, intent, and results/feedback paired with `/teaser-stats`. So we learn, not just blast. |

**Boundary:** these are drafts for Sven's redline where marked. `concept.md` and the ratified sections of `vision-world-company.md` are settled; `business-model.md`, `fundraising.md`, and `go-to-market.md` are v1 captures to be sharpened.
